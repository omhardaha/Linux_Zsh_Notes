### Setup

Type script first compile into Javascript then run.

To Compile `.ts` into `.js` we need to Install Some `typescript` Package.

```bash
npm install -g typescript
```

To compile that run

```bash
tsc file-name.ts
```

- it generate the `.js` file.


Then Finally run the file

```bash
node file-name.js
```

### Another Method
Use package `tsx`.

```bash
npm i -g tsx
```

Then Finally run the file

```bash
tsx file-name.js
```
