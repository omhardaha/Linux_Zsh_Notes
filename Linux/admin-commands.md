### 1. useradd (add user)
```
➜  sudo useradd priyanshu
```
### 2. adduser ( add user diff params )
- can add pw
```
➜  sudo adduser lokesh
```
### 3. su ( change user )
```
➜  su lokesh
```
### 4. exit ( exit user )
```
➜  su lokesh
```
### 5. passwd ( add or change pw )
```
➜  sudo passwd priyamshu
```
### 6. finger ( inspect the user )
- can show login and other info.
```
➜  finger lokesh
```
